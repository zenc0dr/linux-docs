# Операции с файлами



## Копирование

**Копировать (рекурсивно) папку в новое место**

```bash
cp -r /dir/folder1 /dir/folder2
```



**Копировать (рекурсивно) файлы из одной папки в другую**

```bash
cp -r /dir/folder1/* /dir/folder2/
```



**Копировать (рекурсивно) файлы из одной папки в другую вместе с скрытыми файлами**

```bash
cp -r /dir/folder1/{*,.*} /dir/folder2/
```
